Projects

A selection of my data analysis and machine learning projects that demonstrate my technical skills and ability to deliver business value.

  • Movie Profitability Analysis

    Collected and integrated multiple datasets on movies, releases, box office, and metadata. Performed comprehensive analysis using SQL queries and statistical methods.
    Technologies: Python pandas SQL Server Power BI
    Results: Identified key success factors that could increase box office collections by 12%. Delivered actionable recommendations backed by data analysis.

  • Credit Card Default Prediction and Analysis

    Built supervised machine learning models using customer transaction data, payment history, and demographics. Conducted exploratory data analysis to identify correlations and patterns.
    Technologies: Python Scikit-learn XGBoost pandas
    Results: Achieved 0.77 ROC-AUC score, enabling proactive risk management. Model identified key default indicators that could potentially save $1.2M annually for every $10M in credit extended.

  • Hospital Readmission Prediction & Analysis

    Analyzed 100,000+ patient records to identify key risk factors. Developed interactive dashboard for risk visualization and created ML models to predict readmission risk.
    Technologies: Python scikit-learn Dash XGBoost
    Results: Demonstrated potential cost savings of $450,000 per 1,000 patients through targeted interventions based on model predictions.

  • E-commerce Customer Behavior Analysis

    Analyzed 100,000+ transactions across 32,000+ products. Developed interactive Power BI dashboard revealing customer segments and purchasing patterns.
    Technologies: SQL Power BI Python pandas
    Results: Identified 4 high-value customer segments accounting for 70% of revenue. Implemented targeted retention strategies with potential to increase repeat purchase rate by 53% (3.4% to 5.2%).
